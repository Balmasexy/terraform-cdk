 StringConcat <a name="StringConcat" id="cdktf.StringConcat"></a>

- *Implements:* <a href="#cdktf.IFragmentConcatenator">IFragmentConcatenator</a>

Converts all fragments to strings and concats those.

Drops 'undefined's.
