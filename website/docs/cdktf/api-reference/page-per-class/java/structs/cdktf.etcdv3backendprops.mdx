 EtcdV3BackendProps <a name="EtcdV3BackendProps" id="cdktf.EtcdV3BackendProps"></a>

Stores the state in the etcd KV store with a given prefix.

This backend supports state locking.

Read more about this backend in the Terraform docs:
https://www.terraform.io/language/settings/backends/etcdv3

# Initializer <a name="Initializer" id="cdktf.EtcdV3BackendProps.Initializer"></a>

```java
import com.hashicorp.cdktf.EtcdV3BackendProps;

EtcdV3BackendProps.builder()
    .endpoints(java.util.List<java.lang.String>)
//  .cacertPath(java.lang.String)
//  .certPath(java.lang.String)
//  .keyPath(java.lang.String)
//  .lock(java.lang.Boolean)
//  .password(java.lang.String)
//  .prefix(java.lang.String)
//  .username(java.lang.String)
    .build();
```
