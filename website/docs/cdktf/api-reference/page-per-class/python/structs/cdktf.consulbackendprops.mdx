 ConsulBackendProps <a name="ConsulBackendProps" id="cdktf.ConsulBackendProps"></a>

Stores the state in the Consul KV store at a given path. This backend supports state locking.

Read more about this backend in the Terraform docs:
https://www.terraform.io/language/settings/backends/consul

# Initializer <a name="Initializer" id="cdktf.ConsulBackendProps.Initializer"></a>

```python
import cdktf

cdktf.ConsulBackendProps(
  access_token: str,
  path: str,
  address: str = None,
  ca_file: str = None,
  cert_file: str = None,
  datacenter: str = None,
  gzip: bool = None,
  http_auth: str = None,
  key_file: str = None,
  lock: bool = None,
  scheme: str = None
)
```

```python
cert_file: str
```

- *Type:* str

(Optional) A path to a PEM-encoded certificate provided to the remote agent;

requires use of key_file.

---

## `datacenter`<sup>Optional</sup> <a name="datacenter" id="cdktf.ConsulBackendProps.property.datacenter"></a>

```python
datacenter: str
```

(Optional) The datacenter to use.

---

## `gzip`<sup>Optional</sup> <a name="gzip" id="cdktf.ConsulBackendProps.property.gzip"></a>

```python
gzip: bool
```

- *Type:* bool

(Optional) true to compress the state data using gzip, or false (the default) to leave it uncompressed.

---

## `http_auth`<sup>Optional</sup> <a name="http_auth" id="cdktf.ConsulBackendProps.property.httpAuth"></a>

```python
http_auth: str
```

- *Type:* str

(Optional) HTTP Basic Authentication credentials to be used when communicating with Consul, in the format of either user or user:pass.

---

## `key_file`<sup>Optional</sup> <a name="key_file" id="cdktf.ConsulBackendProps.property.keyFile"></a>

```python
key_file: str
```

- *Type:* str

(Optional) A path to a PEM-encoded private key, required if cert_file is specified.

---

## `lock`<sup>Optional</sup> <a name="lock" id="cdktf.ConsulBackendProps.property.lock"></a>

```python
lock: bool
```

- *Type:* bool

(Optional) false to disable locking.

This defaults to true, but will require session permissions with Consul and
at least kv write permissions on $path/.lock to perform locking.

---

## `scheme`<sup>Optional</sup> <a name="scheme" id="cdktf.ConsulBackendProps.property.scheme"></a>

```python
scheme: str
```

- *Type:* str

(Optional) Specifies what protocol to use when talking to the given address,either http or https.

SSL support can also be triggered by setting then environment variable CONSUL_HTTP_SSL to true.
