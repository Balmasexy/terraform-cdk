 ResolveOptions <a name="ResolveOptions" id="cdktf.ResolveOptions"></a>

Options to the resolve() operation.

NOT the same as the ResolveContext; ResolveContext is exposed to Token
implementors and resolution hooks, whereas this struct is just to bundle
a number of things that would otherwise be arguments to resolve() in a
readable way.

# Initializer <a name="Initializer" id="cdktf.ResolveOptions.Initializer"></a>

```go
import "github.com/hashicorp/terraform-cdk-go/cdktf"

&cdktf.ResolveOptions {
	Resolver: github.com/hashicorp/terraform-cdk-go/cdktf.ITokenResolver,
	Scope: github.com/aws/constructs-go/constructs/v10.IConstruct,
	Preparing: *bool,
}
```
